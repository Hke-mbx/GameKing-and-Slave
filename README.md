# GameKing-and-Slave
欢迎来到《王与奴》！ 这是一款中世纪风格的1v1博弈对战型卡牌游戏。玩家操控国王方或奴隶方，使用策略，与对手进行心理战，争取游戏胜利。游戏时长仅在2~6分钟

-----------------------------------------------------------------------------------------------------------------------------------------------
序章 — 王冠与锁链
王权与枷锁在同一桌上较量。两方对弈：王方（维护王室秩序）与奴隶方（背负反抗之名）。每一张出牌都是一次博弈，每一次交锋都可能改写结局。胜利只有两种：王方守住关键，或奴隶方斩落王冠。
 

🎴 一、世界观
========================

在中世纪的王国中，权力与反抗共存。

一方是守护王座的国王方，

一方是渴望推翻统治的奴隶方。

他们的命运，在一场场出牌的心理战中决定。

⚙️ 二、阵营设定
========================
👑 国王方（拥有关键牌：国王、女王、公主）（公主开局开发中）

🩸 奴隶方（目标：摧毁国王方关键牌）

开局选择：

-国王方需选择一种开局方式：国王开局 / 女王开局 / 公主开局
（决定关键牌出场顺序与技能效果）


| 阵营     | 描述             | 关键牌                    | 胜利目标                 |
| ------ | -------------- | ---------------------- | -------------------- |
| 👑 国王方 | 统治者阵营，负责守护权位   | **国王 / 女王 / 公主**（三选一开局）  | 奴隶方失去所有手段（无法击杀关键牌）即胜 |
| 🩸 奴隶方 | 反叛阵营，目标是推翻权力象征 | 无 | 成功击败国王方的关键牌即胜        |


🧭 胜利条件（通用）
========================

王方胜利条件：
-
-奴隶方无法再击杀国王（或女王/公主等关键牌）时，即王方胜利。

奴隶方胜利条件：
-
-王方的关键牌阵亡，即奴隶方胜利。

🪶 三、游戏流程
========================

1.准备阶段
-
-双方确认阵营（国王方 / 奴隶方）。

-确认后给予固定卡牌组

2.出牌阶段
-
-双方各自选择一张牌，背面朝上放在桌上。

-同时翻开。

3.战斗阶段
-
-若两张牌有等级差异 → Level 高者胜，低者阵亡。

-若双方 Level 相同 → 双亡。

-若出现特殊能力（刺客、屠夫） → 按能力优先结算。

4.结算阶段
-
-阵亡牌进入「废牌区」。

-存活牌回到手牌。

-若关键牌阵亡 → 游戏立即结束。


🕹️ 四、开局选择
========================

👑 国王开局（King Opening）
-

-开局拥有以下牌组（共 6 张）：

市民 ×2
刺客 ×1
屠夫 ×1
皇家守卫 ×1
国王 ×1


| 名称                    | 等级 | 类型  | 特殊效果              | 一句话总结            |
| --------------------- | -- | --- | ----------------- | ---------------- |
| 👨 市民（Citizen）     | 1  | 基础牌 | 无            | 「无名无势，却仍忠于王法。」   |
| 🔪 刺客（Assassin）       | 2  | 基础牌 | 无视等级抽牌刺杀 / 刺杀场上目标 | 「黑夜无声，刀光即判生死。」   |
| ⚒️ 屠夫（Butcher）        | 2  | 基础牌 | 无差别同归于尽           | 「血债血偿，是我唯一的信念。」  |
| 🛡️ 皇家守卫（Royal Guard） | 2  | 特殊牌 | 阵亡后下一回合国王免死       | 「陛下在前，生死在后。」     |
| 👑 国王（King）           | 3  | 关键牌 | 阵亡则游戏结束      | 「朕可赦万人，唯独赦不得叛者。」 |

🎯 战略意义
-
👑 优势：
-
-拥有较强的关键牌「国王」，能在出牌上取得主动。
-国王能直接击杀低阶牌（如市民或守卫），在适当时机出国王，可制造手牌数量优势，掌握节奏。


⚔️ 劣势：
-
-国王在开局时已存在于手牌中，代表他始终面临被识破与刺杀的风险。
-若守卫阵亡或刺客未能制衡敌方，国王随时可能被暗杀或屠夫同归于尽。
-一旦被击杀，瞬间暴毙即判输局。

「国王的强大，是建立在所有护卫仍存的前提下。」

================================================
-

-开局拥有以下牌组（共 6 张）：

市民 ×2
刺客 ×1
屠夫 ×1
终极守卫 ×1
女王 ×1

| 名称                       | 等级 | 类型  | 特殊效果              | 一句话总结                   |
| ------------------------ | -- | --- | ----------------- | ----------------------- |
| 👨 市民（Citizen）        | 1  | 基础牌 | 无                                 | 「她的法令是我们的信仰，我们以生命捍卫。」   |
| 🔪 刺客（Assassin）          | 2  | 基础牌 | 无视等级抽牌刺杀 / 刺杀场上目标            |「刀锋触及喉咙，才知生死。」    |
| ⚒️ 屠夫（Butcher）           | 2  | 基础牌 | 无差别同归于尽                            | 「正义与邪恶的血流到地上，都是红的。」 |
| 🛡️ 终极守卫（Ultimate Guard） | 3  | 特殊牌 | 优先斩杀刺客。与女王同生死。不能连续打出。| 「为她挡下最后一剑的，不止忠诚，还有信念。」  |
| 👑 女王（Queen）             | 1  | 关键牌 | 不可打出，阵亡后获得国王                  | 「她的温柔并非怜悯，而是最锋利的审判。」    |

🎯 战略意义
-

-优势：
-

-终极守卫战斗力极强但不能连续打出，给予较强的中局压制力。
-被刺客刺杀（抽中）后，还能召回国王，形成稳健而可控的节奏。
-女王虽无法主动取胜，却以死亡换取平衡与反击，确保局势延续。
-节奏稳健，防御力高。

-劣势：
-
-终极守卫不可连续出动。
-女王仅能最后登场，行动受限。
-一旦终极守卫阵亡，防线骤弱。

「终极守卫将战胜一切，与荣誉共生死」

🎯 战略意义
-

-

-
-缺乏防御型关键牌。
-依赖心理战与出牌顺序。
-奴隶虽能反王，但极易被市民压制。

「卑微的奴隶，亦有推翻王座的机会。」

⚜️ 五、特殊机制
================================================
🛡️ 终极守卫
-
| 属性   | 内容                   |
| ---- | -------------------- |
| 类型   | 特殊牌                  |
| 阵营   | 国王方（女王开局限定）          |
| 等级   | **3**                |
-能击杀任何敌方卡牌。
-不能连续打出。
-若对方是刺客，则刺客的技能失效（无法选择或抽牌），直接被终极守卫击杀。
若对方是屠夫，则屠夫发动技能，双方同时阵亡。
-若女王死亡，终极守卫也会同时死亡。

| 特殊说明 | 拥有强大战力与保护力，是女王开局的“中期壁垒”。其不可连续出动的限制使其战术节奏更有深度。 |

「王权的最后壁垒，忠诚至死。」

-
| 属性     | 内容                                                                      |
| ------ | ----------------------------------------------------------------------- |
| 类型   | 关键牌                  |
| 等级     | 1                                                      |

-女王死亡后，终极守卫**一同殉职**。    |

| 特殊说明   | 象征王权的传承，女王以牺牲让国王登场，形成终局逆转的可能。                                           |

「她的陨落，唤醒了真正的王权。」
