# GameKing-and-Slave
欢迎来到《王与奴》！ 这是一款中世纪风格的1v1博弈对战型卡牌游戏。玩家操控国王方或奴隶方，使用策略，与对手进行心理战，争取游戏胜利。游戏时长仅在2~6分钟

-----------------------------------------------------------------------------------------------------------------------------------------------
序章 — 王冠与锁链
王权与枷锁在同一桌上较量。两方对弈：王方（维护王室秩序）与奴隶方（背负反抗之名）。每一张出牌都是一次博弈，每一次交锋都可能改写结局。规则既要诗意，又必须冷酷无情——胜利只有两种：王方守住关键，或奴隶方斩落王冠。

======================== 

🎴 一、世界观

在中世纪的王国中，权力与反抗共存。
一方是守护王座的国王方，
一方是渴望推翻统治的奴隶方。
他们的命运，在一场场出牌的心理战中决定。

⚙️ 二、阵营设定

1.准备阶段

阵营选择：

两名玩家分别选择操控阵营：
👑 国王方（拥有关键牌：国王、女王、公主）
🩸 奴隶方（目标：摧毁国王方关键牌）

开局选择：
-国王方需选择一种开局方式：国王开局 / 女王开局 / 公主开局
（决定关键牌出场顺序与技能效果）

发牌：
-双方各获得一组卡牌

| 阵营     | 描述             | 关键牌                    | 胜利目标                 |
| ------ | -------------- | ---------------------- | -------------------- |
| 👑 国王方 | 统治者阵营，负责守护权位   | **国王 / 女王 / 公主**（三选一开局）  | 奴隶方失去所有手段（无法击杀关键牌）即胜 |
| 🩸 奴隶方 | 反叛阵营，目标是推翻权力象征 | **奴隶首领**（Slave Leader） | 成功击败国王方的关键牌即胜        |

🧭 胜利条件（通用）

王方胜利条件：
-奴隶方无法再击杀国王（或女王/公主等关键牌）时，即判定王方胜利。

奴隶方胜利条件：
-王方的关键牌全部失效或阵亡，即判定奴隶方胜利。

🪶 三、游戏流程
1.准备阶段
-双方确认阵营（国王方 / 奴隶方）。
-确认后给予固定卡牌组

2.出牌阶段
-双方各自选择一张牌，背面朝上放在桌上。
-同时翻开。

3.战斗阶段
-若两张牌有等级差异 → Level 高者胜，低者阵亡。
-若双方 Level 相同 → 双亡。
-若出现特殊能力（刺客、屠夫、守卫） → 按能力优先结算。

4.结算阶段
-阵亡牌进入「废牌区」。
-存活牌回到手牌。
-若关键牌阵亡 → 游戏立即结束。


🕹️ 四、开局选择

👑 国王开局（King Opening）

游戏流程

1.准备阶段

-双方确认阵营（国王方 / 奴隶方）。
-国王方会有以下牌组（共 6 张）：

市民 ×2
刺客 ×1
屠夫 ×1
皇家守卫 ×1
国王 ×1

2.出牌阶段

-双方各自选择一张牌，背面朝上放在桌上。
-同时翻开。

3.战斗阶段

-若两张牌有等级差异 → Level 高者胜，低者阵亡。
-若双方 Level 相同 → 双亡。
-若出现特殊能力（刺客、屠夫、守卫） → 按能力优先结算。
-结算阶段
-阵亡牌进入「废牌区」。
-存活牌回到手牌。
-若国王阵亡 → 游戏立即结束。

| 🔪 刺客（Assassin）       | 2  | 基础牌 | 无视等级刺杀 / 抽牌；杀屠夫双亡 | 「黑夜无声，刀光即判生死。」   |

⚰️ 特殊规则
-若国王阵亡 → 游戏立即结束，国王方失败。
-若皇家守卫在前一回合阵亡 → 国王获得一回合【免死】保护。

🧠 战术哲学
-国王的力量不在出手，而在掌控全局。
-他身边的人如何行动，决定了他的命运。

🧠 国王开局的战略意义

国王开局（King Opening） 是一场关于“力量与风险”的博弈。

👑 优势：
-拥有较强的关键牌「国王」，能在出牌上取得主动。
-国王能直接击杀低阶牌（如市民或奴隶守卫），在适当时机出国王，可制造手牌数量优势，掌握节奏。

⚔️ 劣势：
-国王在开局时已存在于手牌中，代表他始终面临被识破与刺杀的风险。
-若守卫阵亡或刺客未能制衡敌方，国王随时可能被暗杀或屠夫同归于尽。
-一旦被击杀，瞬间暴毙即判输局。

「国王的强大，是建立在所有护卫仍存的前提下。」

🧠 战略意义：女王开局

-终极守卫战斗力极强但不能连续打出，给予较强的中局压制力。
-被刺客刺杀（抽中）后，还能召回国王，形成稳健而可控的节奏。
-女王虽无法主动取胜，却以死亡换取平衡与反击，确保局势延续。

-优势：
终极守卫战力极强，可稳固中局。
被刺客刺杀后仍能召回国王，维持局势。
女王死亡后国王立即上阵，形成即时逆转机会。
节奏稳健，防御力高。

-劣势：
守卫不可连续出动。
女王仅能最后登场，行动受限。
一旦守卫阵亡，防线骤弱。

💬 一句话总结：
【终极守卫为盾，国王为矛。女王掌控的是时间与秩序的平衡。】

🎯 战略意义

奴隶方拥有数量与博弈优势，虽缺乏高等级单位，但奴隶能够击败国王的设定，使其具备“低位反转”的胜利潜力。
这是最均衡的开局，面对王方的任何变体都有应对空间。

-优势：
拥有刺客与屠夫的中期威胁。

-劣势：
缺乏防御型关键牌。
依赖心理战与出牌顺序。
奴隶虽能反王，但极易被市民压制。

💬 一句话总结：
「卑微的奴隶，也有推翻王座的机会。」

🛡️ 皇家守卫（Royal Guard）机制

特性：⚔️ 【忠诚护主】

规则 ：
皇家守卫的职责是保护国王的安全。
当他阵亡后，会在下一个回合生效一次“国王免死效果”。

💠 详细说明

1️⃣ 战斗时：
-皇家守卫照常参与战斗。
-若胜出 → 回到手牌。
-若阵亡 → 触发「忠诚护主」状态。

2️⃣ 忠诚护主状态：

-从守卫阵亡后开始计时，持续 1 个回合。
-在此期间，若敌方击杀国王（或抽中国王），则：该击杀或抽牌 无效（守卫替国王挡刀）。
-国王幸存。
-该保护效果 仅生效一次，用后即失效。

🧠 一句话总结
【哪怕死去，也要替国王挡下最后一刀。】

⚒️ 屠夫机制（Butcher Mechanism）

特性 ： 血肉同归（Mutual Death）

☠️ 【血肉同归】规则
-不论对方打出什么牌、等级是多少，只要屠夫出现在战斗中，双方的牌都同时阵亡。

⚔️ 详细说明

1️⃣ 当屠夫被打出时：
-若对方牌不是屠夫 → 双方的出牌全部进入废牌区。
-若对方也是屠夫 → 依然双亡（照常处理）。
-若对方为关键牌（如国王、女王、公主） → 关键牌同样阵亡。

2️⃣ 不受等级影响：
-屠夫的效果与等级无关。
-他可以和国王、守卫、刺客、甚至奴隶同归于尽。

3️⃣ 特殊交互：
-若屠夫对上刺客：双方都阵亡（符合“屠夫规则”和“刺客暴露必死”）。
-若屠夫对上市民：双方都阵亡。
-若屠夫对上守卫、国王、关键牌：双方都阵亡。
-若屠夫对上奴隶：双方都阵亡。

🧠 一句话总结
【他不求胜，只求血债。】

🗡️ 刺客机制（Assassination System）

🎴 触发条件

当任意一方打出 刺客（Assassin） 时，进入【暗杀阶段】。

⚔️ 暗杀阶段流程
🩸 情况 A：双方都出刺客

→ 双方同时发动暗杀。

-双方各自 随机抽取对方的一张手牌。
-被抽中的牌立即 阵亡（进入废牌区）。
-若抽中关键牌（国王 / 女王 / 公主） → 立即结束游戏，对方失败。
-暗杀结束后，双方刺客 同时阵亡（暴露即死）。

🌀 结果： 双方都失去 1 刺客 + 各损 1 随机手牌。

🗡️ 情况 B：我方出刺客，对方出非刺客

我方可选择以下两种行动之一👇

1️⃣ 击杀眼前牌（直接刺杀）

-刺客可无视等级，直接击杀眼前对方的出牌。
-但有一个例外：
-若对方出的牌是 屠夫（Butcher），则 双方同时阵亡。
-屠夫拥有“血肉同归”特性（无论谁挑战，都会两败俱伤）。

2️⃣ 潜行暗杀（随机抽牌）

-放弃刺杀眼前牌，改为从对方 手牌中随机抽取 1 张牌。
-被抽中的牌立即阵亡（弃置）。
-若抽中关键牌（国王 / 女王 / 公主） → 立即结束游戏，对方失败。
-对方此回合打出的那张牌 安全返回手牌。

🌀 结果总结：

-刺客能无视等级，主动击杀任何敌方出牌。
-但若对方是屠夫 → 双亡。
-若选择潜行 → 赌运气抽关键牌。

🧠 一句话总结
【刺客可以刺杀任何人，唯独屠夫的刀，会和他一起坠入地狱。】
【刺客的刀，要么刺中国王的心，要么刺穿自己的影子。】
【刺客不靠力量取胜，而靠赌命的机会。】
